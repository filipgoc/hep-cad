Boolean Operations Opérations booléennes

By using Boolean operations , you can create new paths from multiple selected objects. En utilisant des opérations booléennes, vous pouvez créer de nouveaux chemins à partir de plusieurs objets sélectionnés. This works just like it works in maths: path + path = new path . Cela fonctionne comme cela fonctionne en mathématiques: path + path = new path . The original paths are gone after the operation, instead there is a new path (or several), which has been created through the calculation. Les trajets originaux sont passés après l'opération, mais il existe un nouveau chemin (ou plusieurs) créé par le calcul. The stacking order of the original paths determines the result. L'ordre d' empilement des chemins originaux détermine le résultat.

These operations allow to turn multiple paths into a single one, or to break one path up into multiple paths. Ces opérations permettent de transformer plusieurs chemins en une seule, ou de diviser un chemin vers plusieurs chemins. To best understand what exactly the effect of each operation is and in which situations it can be employed most efficiently, you will need to look at them directly . Pour mieux comprendre ce qu'est exactement l'effet de chaque opération est et dans quelles situations il peut être utilisé le plus efficacement possible , vous aurez besoin de les regarder directement. At the moment, it's not important to memorize each detail. Pour le moment, il n'est pas important de mémoriser chaque détail. This is more about knowing that these things exist. Ceci est plus de savoir que ces choses existent. If in doubt, you can simply try it out – and clear up 'accidents' by using the undo feature . En cas de doute, vous pouvez tout simplement essayer - et éclaircir en utilisant la fonction d'annulation des « accidents ». All Boolean operations can be found in the second section of the 'Path' menu. Toutes les opérations booléennes se trouvent dans la deuxième section du menu 'Chemin'.

Difference punches holes that have the shape of the top path into the bottom path. Différence poinçonne des trous qui ont la forme de la trajectoire supérieure à la trajectoire inférieure. This is why this operation only works (as of Inkscape 0.91) with exactly 2 selected paths. C'est pourquoi cette opération ne fonctionne que (Inkscape 0.91) avec exactement 2 chemins sélectionnés.

Intersection Intersection

After using intersection , only those areas are left that have been covered by each one of the selected paths. Après avoir utilisé l' intersection, seules les zones sont laissées qui ont été couverts par chacun des chemins sélectionnés.

Exclusion, Division, Cut Path, Break Apart Exclusion, Division, Cut Path, Break Apart

Exclusion will only keep those areas that do not overlap . L' exclusion ne conservera que les zones qui ne se chevauchent pas. Division will cut a path along the borders of the top path, and creates closed paths. Division coupera un chemin le long des frontières du chemin supérieur, et crée des chemins fermés. Cut path cuts the bottom path's outline in those places, where it intersects the top path . Chemin coupé coupe le contour du chemin bas dans ces endroits, où il croise le chemin en haut. Break apart will break a path up into its subpaths , where each subpath will become a new object. Scinder se frayer un chemin vers le haut dans ses sous - chemins, où chaque sous - chemin deviendra un nouvel objet.
